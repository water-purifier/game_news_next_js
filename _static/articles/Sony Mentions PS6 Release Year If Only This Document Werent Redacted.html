索尼提到PS6发布年，如果仅此文档已编辑

2022年11月24日

在其最新的CMA对微软计划收购Activision Blizzard的回应之一（被游戏行业分析师Piers Harding-Rolls发现）中，索尼提到了其下一代PlayStation游戏机以及如何失去获得“使命召唤”系列的机会，这将带来重大影响 利润。 虽然简短的文本线没有提及由于其中的修订而提及的确切日期，但它确实确认索尼计划了将来发布的新PlayStation，可能是在2027年或稍后再稍后发布。

索尼的评论写道：“微软已提出要继续在PlayStation上提供Activision的游戏，直到2027年。” “同样，在10月26日的公开评论中，微软表示，它计划仅在有意义的情况下才能在PlayStation上提供使命召唤。直到2027年或其他（可能更短的时间）Microsoft单方面决定的时期 对微软的意识是不足的。到SIE推出下一代PlayStation控制台（可能发生在删除的情况下）时，它将无法访问《使命召唤》和其他激活标题，这使其非常容易受到消费者的攻击 切换和随后的竞争力下降。”

PlayStation控制台的保质期通常为6 - 7年，当前的模型PlayStation 5庆祝了本月的第二个生日。 索尼在PS5生命周期的现阶段重点关注PS6控制台并不罕见，因为Xbox和Nintendo都实践了恒定的控制台研发。

索尼一直是反对微软收购的最大声音之一，它标有“改变游戏规则的人，对数亿消费者享受的行业构成威胁”。 索尼一直专注于可能失去获利的《使命召唤》系列，而微软游戏首席执行官菲尔·斯潘德（Phil Spender）经常通过重申Microsoft对Activision的移动游戏目录更感兴趣，从而定期低估了这些恐惧。

Spencer在10月指出，该公司计划像Minecraft这样的Minecraft这样的职责召唤，并回应了Sony对Microsoft未来对《使命召唤》的所有权的批评。 Spencer说：“ Activision的想法是关于主机上的“使命召唤”的想法，是我们的游戏机竞争对手可能创建的构造。”
