FTC可能会通过Microsofts Activision暴风雪收购报告提起反托拉斯诉讼

2022年11月24日

当微软在世界各地的监管机构试图获得Activision Blizzard方面面临审查时，它可能会遇到其在美国迄今为止最重要的障碍之一。 Politico的一份新报告显示，联邦贸易委员会“可能”就拟议交易提起反托拉斯诉讼。

据说诉讼尚不保证，因为FTC的专员尚未对此举进行投票，并且可能首先希望与Microsoft和Activision Blizzard的律师会面。 FTC的调查正在进行中，但其中大部分已经完成，并且可能在12月提起诉讼。 FTC拒绝对这种可能性发表评论。

微软在1990年代就面临着美国政府的重大反托拉斯诉讼，但在近年来，即使在向像亚马逊，Google和Facebook等技术巨头施加压力的同时，这些问题也更少。 但是，过去处理反托拉斯投诉的新FTC主席莉娜·汗（Lina Khan）可能会寻求退缩拟议的690亿美元收购Activision Blizzard，这将是Microsoft有史以来最大的一笔。

在此之前，英国的竞争和市场管理局已经达到了微软的交易。 尽管尚未宣布决定，但CMA表示，对于该交易可能“大大减轻竞争”的销售，对游戏机，订阅服务和云游戏网络的销售感到“担心”。

如果这笔交易通过，索尼将成为受到影响的主要公司之一，并且竭尽全力声称收购会严重伤害它。 除其他事项外，索尼对《使命召唤》成为Xbox独家的前景引起了警报，尽管据报道，微软已采取了不寻常的提议，以确保该系列将在PlayStation平台上保留十年。 微软还肯定会强调，如果第三名游戏机制造商比一个游戏系列获得排他性，PlayStation当前的市场优势将无法完全消除。

虽然加强Xbox Game Pass是Microsoft达成交易动机的一部分，但它说的主要因素是在日益有利可图的移动游戏领域中获得立足点。
